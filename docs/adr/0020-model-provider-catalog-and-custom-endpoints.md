# Model provider catalog and custom endpoints

Minimalist Agent will support multiple model providers through Model Configurations rather than limiting the platform to one official provider. The Administrator Console will include a Model Provider Catalog with common providers, display logos, default endpoint templates, and documentation links, while also allowing Administrators to create a Custom Model Endpoint for a self-hosted gateway, proxy, or OpenAI-compatible relay. Agents select from enabled Model Configurations and never receive raw credentials or provider-specific secrets directly.
