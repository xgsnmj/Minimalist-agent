# Search capability with Doubao default

Minimalist Agent will expose web search as Search Capability with provider configuration rather than hard-coding search behavior into Agent logic. The MVP default implementation will use Doubao Search Provider, while the Agent Conversation UI, Agent Event Protocol, and Agent Tool Gateway remain provider-agnostic. Administrators can enable Search Capability per Agent and configure provider credentials, availability, limits, and operational settings from the Administrator Console; page reading is configured separately as Page Read Capability.
