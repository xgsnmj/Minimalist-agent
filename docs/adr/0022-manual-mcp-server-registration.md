# Manual MCP server registration before marketplace

Minimalist Agent will support administrator-managed MCP Server registration in the MVP and will not build an MCP marketplace. Administrators can configure supported MCP connection types, credentials or environment variable references, enablement state, discovery checks, and MCP Tool Authorization per Agent. A marketplace would add source trust, review, installation, updates, versioning, and dependency management before the platform has proven its core Agent Conversation and tool governance workflows.
