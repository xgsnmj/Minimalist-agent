# Administrator console scope

Minimalist Agent's MVP Administrator Console will focus on platform governance rather than analytics. It will include account approval, Agent lifecycle and capability policy, Model Configuration, MCP Server registration and tool authorization, Search Provider Configuration, Page Read Provider Configuration, sandbox status, and Run Audit with Full Trace access. A lightweight overview can show pending accounts, recent failed runs, provider health, and storage usage, but a BI-style dashboard is outside the MVP scope.
