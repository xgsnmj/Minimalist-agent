# Conversation list before projects

Minimalist Agent will make Agent Conversations the first-level navigation object in the MVP and will not introduce projects, spaces, folders, or collaboration containers initially. The WorkBuddy-like sidebar can include new conversation, search or filtering, recent conversations, settings, and Administrator Console entry points, while long-lived project organization remains a later extension. This keeps the information architecture aligned with the single-workspace MVP and avoids adding ownership or collaboration concepts before they are needed.
