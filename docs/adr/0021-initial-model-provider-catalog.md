# Initial model provider catalog

Minimalist Agent will seed the Model Provider Catalog with OpenAI, Anthropic, Google Gemini, DeepSeek, Qwen or Alibaba Cloud DashScope, Moonshot or Kimi, ByteDance Doubao, Zhipu or GLM, MiniMax, OpenRouter, and Custom OpenAI-compatible endpoint entries. Catalog entries provide display metadata such as name, logo, endpoint template, compatibility notes, recommended model examples, and documentation links; they do not make a provider usable until an Administrator supplies credentials and enables a Model Configuration.
